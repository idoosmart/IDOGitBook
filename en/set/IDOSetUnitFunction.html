weightUnit weight unit 0x00: invalid, 0x01: kg, 0x02: lb, 0x03: Ying Shi
tempUnit Temperature unit 0x00: invalid, 0x01: °C, 0x02: °F

Menu judgment
strideWalk Walking pace According to male conversion, the default value is 72cm
strideRun Run pace According to male conversion, the default value is 90cm
strideGps gps calibration step size 0x00: invalid, 0x01: on, 0x02: off
timeUnit Time unit 0x00: invalid, 0x01: 24-hour clock, 0x02: 12-hour clock
weekStart The start day of the week Sunday: 0x01, Monday: 0x00, Saturday: 0x03

Command code

Objc:

IDOSetUnitInfoBluetoothModel * model = [IDOSetUnitInfoBluetoothModel currentModel];
[IDOFoundationCommand setUnitCommand:model
                            callback:^(int errorCode) {
    if (errorCode == 0) {
       //Set up successfully
    }else if (errorCode == 6) {
      //The device does not support
    }else {
      //Setup failed
    }
}];

Swift:

let model = IDOSetUnitInfoBluetoothModel.current();
IDOFoundationCommand.setUnit(model) {(errorCode) in
     if errorCode == 0 {
        //Set up successfully
     }else if errorCode == 6{
        //The device does not support this method
     }else {
       //Setting error
     }
};

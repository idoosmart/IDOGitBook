Bluetooth upgrade library

Bluetooth upgrade library

3.1 Function overview

The SDK upgrade function is only responsible for firmware upgrades. Firmware version determination and firmware download are not processed. Pay attention to the integrity of the firmware download, pass in the firmware local sandbox path during the upgrade, monitor the upgrade progress and completion status, and error agent callbacks. The current version of the upgrade library adds the nordic, realtk, and apollo platforms. The upgrade management library (IDOBlueUpdate) depends on the Bluetooth protocol library (IDOBlueProtocol), so before using the upgrade management library, first import the Bluetooth protocol library.

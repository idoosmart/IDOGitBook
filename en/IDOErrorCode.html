Error code description

Error code description

5.1Functional Overview

errorCode is the basis for judging the status of the global command callback. When the errorCode is equal to 0, it is successful, and other values are failure. The error code string can be obtained through the method in the IDOErrorCodeToStr class, or the explanation of the code corresponding to the documentation can be used。

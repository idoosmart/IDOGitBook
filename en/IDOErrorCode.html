Error code description

5.1 Function overview

The errorCode error code is the basis for the judgment status of the global command callback. When errorCode is equal to 0, it is successful, and the other values ​​are failure. The error code string can be obtained by the method in the IDOErrorCodeToStr class, or the explanation of the corresponding code in the description document.

  • IDOSyncBloodOxygenDataInfoBluetoothModel
Parameters Description Remarks
year year int
month month int
day date int
dateStr Date accurate to date str
minuteOffset Total time offset int (seconds)
itemsCount Blood oxygen data packet int
bloodOxygens Blood oxygen collections Only defined query methods can be converted into model collections,
custom query methods cannot be directly converted into model collections,
itemModel needs to be queried and assigned to the current attribute
avgBloodOxygen Average blood oxygen int
maxBloodOxygen Maximum blood oxygen int

  • Query blood oxygen by year

Objc:

[IDOSyncSpo2DataModel queryOneYearBloodOxygenWithYear:year
                                              macAddr:macAddr
                                         isQueryItems:NO];

Swift:

IDOSyncSpo2DataModel.queryOneYearBloodOxygen(withYear: year, macAddr: macAddr, isQueryItems: false);
  • Check blood oxygen by month

Objc:

[IDOSyncSpo2DataModel queryOneMonthBloodOxygenWithYear:year
                                                 month:month
                                               macAddr:macAddr
                                          datesOfMonth:&days
                                          isQueryItems:NO];

Swift:

IDOSyncSpo2DataModel.queryOneMonthBloodOxygen(withYear:year, month:month, macAddr:macAddr, datesOfMonth:&days, isQueryItems:false);

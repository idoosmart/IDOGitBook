设备通知APP交换数据暂停

设备通知APP交换数据暂停

App收到的json字段：

字段名 字段类型 字段说明
day int 恢复时间 日
hour int 恢复时间 时
minute int 恢复时间 分
second int 恢复时间 秒

示例：

{
    "day":26,
    "hour":10,
    "minute":56,
    "second":15
}

App下发的json字段:

字段名 字段类型 字段说明
err_code int err_code
0：成功
1：没有进入运动模式失败

示例：

{
    "err_code":0
}

### 设备获取APP登录状态


**App下发的json字段**:

| 字段名    | 字段类型 | 字段说明                                                     |
| --------- | -------- | ------------------------------------------------------------ |
| log_state | int      | 0：正常状态（登录）<br />1：未登录<br />2：网络断连<br />3：未获取状态 |

`示例：`

```c
{
    "log_state":0
}
```

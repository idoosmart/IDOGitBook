### APP设置识别后的状态


功能表:getVoiceTransmission

**App下发的json字段**:

| 字段名      | 字段类型 | 字段说明                                          |
| ----------- | -------- | ------------------------------------------------- |
| phone_state | int      | 0：正常状态<br />1：无法识别<br />2：语音识别超时 |

`示例：`

```c
{
    "phone_state":0
}
```

**App收到的json字段**：

| 字段名     | 字段类型 | 字段说明         |
| ---------- | -------- | ---------------- |
| is_success | int      | 1：成功 ， 0失败 |

`示例：`

```c
{
    "is_success":1
}
```

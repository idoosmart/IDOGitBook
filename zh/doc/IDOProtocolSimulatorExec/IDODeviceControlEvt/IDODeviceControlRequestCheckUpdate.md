### ~~设备请求版本检查~~


**App收到的json字段**：

| 字段名 | 字段类型 | 字段说明 |
| ------ | -------- | -------- |
|        |          |          |

**App下发的json字段**:

| 字段名  | 字段类型 | 字段说明                                                     |
| ------- | -------- | ------------------------------------------------------------ |
| flag    | int      | 0 已经是最新版本,1 有新版本,2 网络错误,3 其他错误 |
| version | int      | 协议版本                                                     |

`示例：`

```c
{
    "flag":0,
    "version":0
}
```

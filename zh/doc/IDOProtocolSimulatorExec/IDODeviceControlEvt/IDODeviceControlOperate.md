### ~~设备通知APP操作类型~~


**App收到的json字段**:

| 字段名  | 字段类型 | 字段说明                                |
| ------- | -------- | --------------------------------------- |
| timeout | int      | 超时时间 单位秒                         |
| type    | int      | 1 短按,2 长按,3 敲打,4 双敲 |

`示例：`

```c
{
    "timeout":30,
    "type":1,
}
```

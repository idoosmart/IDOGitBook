### APP设置ALEXA通知状态


功能表:getVoiceTransmission

**App下发的json字段**:

| 字段名      | 字段类型 | 字段说明                                                  |
| ----------- | -------- | --------------------------------------------------------- |
| notify_flag | int      | 0：无效值<br />1：有通知（黄色）<br />2：通知清除（蓝色） |

`示例：`

```c
{
    "notify_flag":0
}
```

**App收到的json字段**：

| 字段名     | 字段类型 | 字段说明         |
| ---------- | -------- | ---------------- |
| is_success | int      | 1：成功 ， 0失败 |

`示例：`

```c
{
    "is_success":1
}
```
